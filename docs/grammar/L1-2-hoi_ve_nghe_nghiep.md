## **Grammar: The Question Particle 吗 (ma)**

**Definition:** "吗 (ma)" is a modal particle placed at the end of a declarative sentence to turn it into a "yes/no" question.  
`trợ từ ngữ khí`: "吗 (ma)" đặt ở cuối câu trần thuật để tạo thành câu hỏi "có/không".

### **Sentence Structure**

> **Chủ ngữ (Subject) + Động từ (Verb) + Tân ngữ (Object) + 吗 + ?**

### **Examples**

| Chinese (Pinyin) | Vietnamese Translation |
| --- | --- |
| <button onclick="speakNormal('肖明是学生。')">**肖明是学生。**</button> (Xiào Míng shì xuésheng.) | Tiêu Minh là học sinh. |
| <button onclick="speakNormal('肖明是学生吗？')">**肖明是学生吗？**</button> (Xiào Míng shì xuésheng **ma**?) | Tiêu Minh có phải là học sinh không? |
| <button onclick="speakNormal('你叫王一山。')">**你叫王一山。**</button> (Nǐ jiào Wáng Yīshān.) | Bạn tên là Vương Nhất Sơn. |
| <button onclick="speakNormal('你叫王一山 吗？')">**你叫王一山 吗？**</button> (Nǐ jiào Wáng Yīshān **ma**?) | Bạn tên là Vương Nhất Sơn phải không? |
| <button onclick="speakNormal('你学习汉语。')">**你学习汉语。**</button> (Nǐ xuéxí Hànyǔ.) | Bạn học tiếng Trung. |
| <button onclick="speakNormal('你学习汉语 吗？')">**你学习汉语 吗？**</button> (Nǐ xuéxí Hànyǔ **ma**?) | Bạn có học tiếng Trung không? |

## **How to Answer "Yes/No" Questions**

### **1. Simple Answers**

Use the verb from the question. For affirmation, use the **verb** directly. For negation, use **"不 (bù) + verb"**.

* **Question:** <button onclick="speakNormal('你是学生吗？')">你是学生吗？</button> (Nǐ shì xuésheng ma? - Cậu là học sinh phải không?)
* **Affirmative:** <button onclick="speakNormal('是。')">是。</button> (Shì. - Phải.)
* **Negative:** <button onclick="speakNormal('不是。')">不是。</button> (Bú shì. - Không phải.)



### **2. Full/Complete Answers**

Repeat the statement with the correct information.

* **Question:** <button onclick="speakNormal('你学习汉语吗？')">你学习汉语吗？</button> (Nǐ xuéxí Hànyǔ ma? - Bạn học tiếng Trung à?)
* **Affirmative:** <button onclick="speakNormal('我学习汉语。')">我学习汉语。</button> (Wǒ xuéxí Hànyǔ. - Tôi học tiếng Trung.)
* **Negative:** <button onclick="speakNormal('我不学习汉语。')">我不学习汉语。</button> (Wǒ bù xuéxí Hànyǔ. - Tôi không học tiếng Trung.)
